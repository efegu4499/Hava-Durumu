# Hava Durumu Uygulaması

Bu proje, Open-Meteo API'sini kullanarak şehir bazlı hava durumu sorgulayan küçük bir Python uygulamasıdır.

## Çalıştırma

### Arayüzlü çalışma

```bash
python main.py
```

Bu komut, görsel arayüzü açar.

### Konsol sorgusu

```bash
python main.py Istanbul
```

İsterseniz şehir adını komut satırından girebilir veya arayüz üzerinden sorgulayabilirsiniz.

## Özellikler

- Şehir bazlı hava durumu sorgusu
- Sıcaklık, nem ve rüzgar bilgisi
- Görsel arayüz ve hava görünümü
- Açık / kapalı / yağmurlu gibi açıklayıcı durumlar
